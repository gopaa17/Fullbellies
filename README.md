🍽️ FullBellies – Zero Hunger, Zero Waste:
FullBellies is a tech-driven platform that connects surplus food from restaurants, hotels, and cloud kitchens to people in need. Our mission is to reduce food waste while ensuring no one sleeps hungry.

📌 Problem Statement:
Every day, tons of food is wasted while millions go hungry. There is a lack of an efficient system to redistribute surplus food safely and quickly.

💡 Solution Overview:
FullBellies bridges this gap by creating a smart platform where:
- Food providers can list surplus meals
- Users/NGOs can request or receive food
- Delivery ensures quick redistribution

This creates a "sustainable, scalable, and impactful food ecosystem".

🚀 Key Features:
- 🍱 Surplus Food Listing (Restaurants & Hotels)
- 🤝 Connects Donors with People in Need
- 🚚 Fast Delivery / Pickup System
- 💚 Pay-it-forward donation model
- 📊 Admin Dashboard for monitoring
- 🔔 Real-time notifications

🧠 Technical Architecture:
Frontend:
- HTML, CSS (TailwindCSS)
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Other Tools:
- Git & GitHub
- VS Code

⚙️ How It Works:
1. Food providers list surplus meals  
2. Platform matches with nearby users/NGOs  
3. Delivery/collection is arranged  
4. Food reaches those in need
